how to start project Laravel project with vue.js typescript and tailwind with Laravel Route, api

composer create-project --prefer-dist laravel/laravel SourceCode

cd SourceCode

npm install vue vue-router axios

npm install -D typescript @vitejs/plugin-vue vite laravel-vite-plugin tailwindcss postcss autoprefixer

npx tailwindcss init -p
